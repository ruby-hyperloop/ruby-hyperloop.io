Support

React.rb is worked on by a team of contributors. Many features and concepts are a result of early adopters like you. Please get in touch, there is so much to do!

Gitter

There is almost always someone available to ask a quick question or discuss a concept on gitter.im

Stack Overflow

Although its early days we encourage you to use stack overflow to ask specific questions. existing questions tagged with react.rb or ask your own!

Github issues

If you think you found a bug, or want to suggest a new feature github is the place to go.

Twitter

For the latest news about React, follow @reactrb on Twitter. In addition, you can use the #reactrb hashtag to see what others are saying or add to the conversation.
